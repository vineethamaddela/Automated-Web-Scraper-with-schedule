# Automated Web Scraper with Scheduler

## Description
This project scrapes quotes and author names from https://quotes.toscrape.com.

## Features
- Scrapes quotes from website
- Saves data to CSV
- Saves data to SQLite database
- Uses Python, Requests and BeautifulSoup

## Technologies
- Python
- Requests
- BeautifulSoup (bs4)
- SQLite
- CSV

## Output Files
- quotes.csv
- quotes.db
